# Visit Attractions - An App for Travel Lovers

## Overview

Users can search attractions in a specific state and get some information 
Users also find the most popular attractions are entirely driven by other users ratings and comments
This app was written in ReactJS as a Capstone Project at Ada Developers Academy.

## App Features
    - Users are able to search a specific state in the Us and find it's attractions
    - users can login and have an account and bookmark the attractions
    - users can add new attractions and new place(state)
    - users can write comments and rate other attractions
    - users are able to delete their posts
    - users have access to all their favorite attractions and their posts in their profile
    - users can see the location of all the attractions on a google map
